## 🔧 How to Use QR-Generator

### Step 1: Select QR Type
Choose between:

- URL: For website links  
- UPI Payment: For payment codes typical in India  

### Step 2: Enter Information
- For URL: Type any website address starting with `http://` or `https://`.  
- For UPI: Enter the required payment details. These usually include your UPI ID and amount.  

### Step 3: Generate Code
Press the "Generate" button. The app creates a QR code image on the screen.  

### Step 4: Save or Scan
- To save, right-click the QR code and choose **Save image as...** on Windows.  
- To scan, open your phone’s camera or UPI app and point at the QR code.
